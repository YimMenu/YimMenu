# Table: globals

Table containing functions for manipulating gta script globals.

## Functions (9)

### `get_int(global)`

Retrieves an int global value.

- **Parameters:**
  - `global` (integer): index of the global

- **Returns:**
  - `integer`: value of the global

**Example Usage:**
